##OS Keychains collection

This is a repository containing some 3D printable logos of different operating systems as seen at [ThingiVerse] (http://www.thingiverse.com/thing:1954171)


- Ubuntu Linux
- Windows 10
- Debian
- Arch Linux
- Fedora

More are to be added.
